# Chat Application

A simple real-time chat application built using Python sockets and threading.

## Features

- Real-time messaging
- Multiple clients
- Server-client architecture
- Message timestamps
- Graceful disconnect
- Localhost communication

## Technologies Used

- Python
- Socket Programming
- Threading

## How to Run

### Start the Server

```bash
python server.py
```

### Start the Client

```bash
python client.py
```

Open multiple client terminals to chat.
