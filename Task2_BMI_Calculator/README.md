# Advanced BMI Calculator

## Description

The Advanced BMI Calculator is a Python GUI application that calculates a user's Body Mass Index (BMI), classifies the result into standard health categories, stores records in an SQLite database, and displays BMI history with graphical visualization.

## Features

- GUI using Tkinter
- Calculate BMI
- BMI Classification (Underweight, Normal, Overweight, Obese)
- Color-coded results
- Multi-user support
- Save BMI records using SQLite
- View BMI history
- BMI trend graph using Matplotlib
- Input validation
- Database error handling

## Technologies Used

- Python
- Tkinter
- SQLite3
- Matplotlib

## Installation

1. Clone the repository

```bash
git clone https://github.com/HimavadanKamera/OIBSIP.git
```

2. Navigate to the project folder

```bash
cd OIBSIP/Task2_BMI_Calculator
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python main.py
```
