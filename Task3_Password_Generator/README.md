# Advanced Random Password Generator

## Description

The Advanced Random Password Generator is a Python GUI application that creates secure and customizable passwords based on user-selected criteria. It provides a simple interface for generating strong passwords suitable for online accounts and applications.

## Features

- Generate secure random passwords
- Select uppercase letters
- Select lowercase letters
- Select numbers
- Select symbols
- Password strength indicator
- Copy password to clipboard
- Exclude ambiguous characters (0, O, l, 1)
- Stores the last 5 generated passwords
- Input validation

## Technologies Used

- Python
- Tkinter
- Secrets
- String
- Pyperclip

## Installation

1. Clone the repository

```bash
git clone https://github.com/HimavadanKamera/OIBSIP.git
```

2. Navigate to the project

```bash
cd OIBSIP/Task3_Password_Generator
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the project

```bash
python main.py
```
